# Tatiana Campo Villalba · Portfolio

Sitio web personal de **Tatiana Campo Villalba**, Product Designer con más de 15 años de experiencia en tecnología y especialista en sistemas de diseño.

Proyecto desarrollado como **segunda entrega** del curso de **Desarrollo Web de CoderHouse**.

🔗 **Repositorio:** https://github.com/tcampovillalba/segunda-entrega-coder-house

---

## 📄 Descripción

Portfolio de diseño de producto que reúne casos de estudio reales, servicios ofrecidos y un sistema de diseño propio. Está construido con **HTML5** y **CSS3**, con maquetado responsive y tipografías de Google Fonts (*Roboto* y *Mynerve*).

## 🗂️ Estructura del sitio

| Página | Archivo | Contenido |
|--------|---------|-----------|
| Inicio | `index.html` | Presentación, hero y grilla de casos de estudio |
| Sobre mí | `sobre-mi.html` | Perfil y trayectoria profesional |
| Servicios | `servicios.html` | Servicios de diseño ofrecidos |
| Contacto | `mi-contacto.html` | Formas de contacto |
| Sistema de diseño | `sistema-de-diseno.html` | Componentes, colores y tipografías del sitio |

### Casos de estudio
`provincia-art.html` · `mandu.html` · `oms.html` · `calipso.html` · `buenos-aires.html` · `ieb.html` · `pas.html` · `preventores.html`

## 📁 Organización de archivos

```
segunda-clase/

├── style.css/

```

## 🚀 Cómo verlo

1. Cloná el repositorio:
   ```bash
   git clone https://github.com/tcampovillalba/segunda-entrega-coder-house.git
   ```
2. Abrí `index.html` en tu navegador (doble clic o con la extensión *Live Server* de VS Code).
